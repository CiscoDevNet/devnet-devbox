# sandbox-devbox
Documentation around the DevNet Sandbox Devbox VM

# Usage

`vagrant up`

# VM Information

* OS: CentOS 7
* Mem: 2GB
* HD: 20GB


# Software

* Gnome Desktop
* Docker Daemon and Engine
* git tools
* Python 2.7 and 3.5
* pip
* virtualenv
* drone command line tool
* PyCharm Community
* node version manager (nvm)
* node v6.9.2 (LTS)
* atom 1.12.7
